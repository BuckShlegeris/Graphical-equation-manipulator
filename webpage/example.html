Graphical equation manipulator

GEM is a work-in-progress program which lets you graphically manipulate physics equations. Here's a screenshot.

Installation

If you're on Windows, you're lucky: just download the .exe from here.

If you're on Linux or a Mac, you have to run it from source. Get the source code from Github. The only dependency is Sympy. Once you've installed Sympy, run main.py in Python 2.
